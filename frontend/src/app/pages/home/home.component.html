Autos2 is a modern vehicle database application built with Angular 14 and powered by Elasticsearch. It provides a fast and intuitive interface for exploring vehicle data, with powerful search and filtering capabilities.

This application demonstrates best practices in modern web development, including component-based architecture, reactive programming, and containerized deployment.
